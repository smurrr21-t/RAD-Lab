Welcome to RAD Lab

This webpage is created for the Rapid Application Development (RAD) lab. The purpose of this project is to practice using Git and GitHub for managing and tracking changes in a software project.

GitHub allows developers to store their code online and collaborate with other developers. It also helps track different versions of a project and manage updates efficiently.

In this lab, we created a repository, uploaded project files, and organized the project structure using folders such as CSS and JavaScript.
